Setup & Configuration

Connect Supabase (Free Backend)

Go to supabase.com and create a FREE account and new project.
In your Supabase project, go to Table Editor → New Table. Name it portfolio_works. Add columns: id (uuid, default: gen_random_uuid()), title (text), category (text), description (text), video_url (text), thumbnail_url (text), client_name (text), tags (text[]), featured (bool, default false), created_at (timestamptz, default: now()).
Go to Storage → New Bucket. Name it portfolio-media and make it Public.
Go to Project Settings → API. Copy the Project URL and anon public key.
Open js/site-config.js and paste the URL and key into supabaseUrl and supabaseAnonKey. Also change the adminPasscode to something strong.
In Supabase → Authentication → Policies, enable Row Level Security and add a policy on portfolio_works allowing SELECT for everyone (anon), and INSERT/UPDATE/DELETE for authenticated or for all (since this is a personal site). For storage, allow all operations on portfolio-media.
Re-deploy your site. The admin panel will now save works to Supabase and let you upload images/videos!
